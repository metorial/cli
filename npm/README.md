# `@metorial/cli`

Install the Metorial CLI with npm:

```bash
npm install -g @metorial/cli
```

The npm package is a lightweight wrapper. On first run it downloads the matching native CLI binary for its own package version from GitHub Releases, caches it locally, and then executes it.

The wrapper version is intentionally locked to the CLI release version, so `@metorial/cli@1.2.3` always downloads the `v1.2.3` Metorial CLI binaries.
